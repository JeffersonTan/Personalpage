Tech

Since I was very young, I like to tinker with tech, and I'm eager to learn about how things work

If I want something to work the way I want it to, I will try to find a way to make it work. Usually I would hunt forum posts or videos online for guides to do things, try it out, and see if things work. But sometimes... that can produce unintended results.

I've probably broken stuff in countless ways by now, some examples are: Bricking a phone, accidentally wiping a drive, and so on, but that's all part of the learning process. If I didn't fail, I wouldn't have the knowledge and experience that I have today.
