What is Umami?

So, what is this Umami thing anyways? Umami is a FOSS (Free and Open-Source) and privacy-focused web analytics uhh.. thing. It helps web admins view visitor data and whatnot for their websites.

If you'd like to learn more (and get a better explanation), you can check out their website here.

Why Umami?

Why not just use Google Analytics or CloudFlare like any other sane person?

Some of you may wonder why I chose this over any other analytics solution like Google Analytics. It's because I care about my privacy and the privacy of visitors like you guys and gals who visit this website, plus, I'm not a big fan of Google anyways.

Another reason is that I can self-host Umami on my home server. I'll probably show my server some other time, but being able to self-host means that I control the data instead of some big corp. If you're curious, you can check out the analytics data over here.

Conclusion

So in short, Umami is an analytics platform that's free and respects users' privacy. I chose it because I don't want to use Google and because I can host Umami myself on my own server.
