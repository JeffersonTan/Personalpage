Tech

Since I was very young, I like to tinker with tech, and I'm eager to learn about how things work

If I want something to work the way I want it to, I will try to find a way to make it work. Usually I would hunt forum posts or videos online for guides to do things, try it out, and see if things work. But sometimes... that can produce unintended results.

I've probably broken stuff in countless ways by now, some examples are: Bricking a phone, accidentally wiping a drive, and so on, but that's all part of the learning process. If I didn't fail, I wouldn't have the knowledge and experience that I have today.

Projects

Below are a few projects that I have done in the past

Mechanic's Guide (2019)

This is a website that I made for a high school project many years ago. It's coded using both HTML and CSS by hand, and since I used GitHub pages to host it, I learned a bit about code versioning.

GPU Detector Script (2024)

This was a small project I've made using Bash to learn more about Bash scripting and as a companion script for my other Bash project

FFmpeg Accel Script (2024)

This is mainly a script that I wrote to help myself convert videos faster and with less effort, but anyone else are free to use it. Uses some code from my GPU detection script. It is currently a work in progress but the basics are there.
